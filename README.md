# Página con menú izquierdo

Arquitectura mínima con una barra lateral izquierda, CSS y un pequeño script para togglear el menú en pantallas pequeñas.

Cómo usar:

1. Abrir `index.html` en tu navegador (doble clic o arrastrar al navegador).
2. En pantallas pequeñas pulsa el botón ☰ para mostrar/ocultar el menú.

Archivos:

- index.html
- styles.css
